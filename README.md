# 🕌 نظام إدارة المدرسة القرآنية

تطبيق سطح مكتب مبني بـ **Electron.js + SQLite** لإدارة المدرسة القرآنية بالكامل.

---

## 📋 المتطلبات

- **Node.js** (الإصدار 18 أو أحدث) — https://nodejs.org/en/download
- **npm** (يأتي مع Node.js تلقائياً)

---

## 🚀 تشغيل التطبيق

افتح المجلد في Terminal أو CMD ثم اكتب:

```bash
# 1. تثبيت المكتبات (مرة واحدة فقط)
npm install

# 2. تشغيل التطبيق
npm start
```

---

## ✨ المميزات

| الوحدة | الوصف |
|--------|-------|
| **لوحة التحكم** | إحصائيات شاملة، أفضل الطلاب، آخر النشاطات |
| **الطلاب** | إضافة / تعديل / حذف، تتبع التقدم بالنسبة المئوية |
| **ملف الطالب** | خريطة حفظ كاملة للـ 114 سورة |
| **الأساتذة** | إدارة الكادر التدريسي وربطهم بالطلاب |
| **تسجيل الحفظ** | إضافة جلسات الحفظ بتقييمات (ممتاز/جيد/مقبول/ضعيف) |
| **الحضور والغياب** | تسجيل يومي مع إحصائيات فورية |
| **التقارير** | تقرير شامل قابل للطباعة لجميع الطلاب |

---

## 🗄️ قاعدة البيانات

يتم حفظ البيانات تلقائياً في ملف `quran_school.db` داخل مجلد بيانات التطبيق:
- **Windows:** `%APPDATA%\quran-school-manager\`
- **Mac:** `~/Library/Application Support/quran-school-manager/`
- **Linux:** `~/.config/quran-school-manager/`
